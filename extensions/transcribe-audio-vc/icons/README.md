# Ícones da Extensão

Para melhor aparência, adicione ícones PNG nos seguintes tamanhos:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Sugestões de Design

- Use um ícone de microfone, ondas sonoras ou áudio
- Cores sugeridas: #667eea, #764ba2 (gradiente roxo/azul)
- Fundo transparente ou sólido

## Ferramentas para Criar Ícones

- [Figma](https://figma.com) - Design profissional
- [Canva](https://canva.com) - Templates prontos
- [Favicon.io](https://favicon.io) - Gerador de ícones
- [Icon Kitchen](https://icon.kitchen) - Específico para extensões
